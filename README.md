### Reinforcement Learning Minesweeper
**Focus:** *Deep RL & Strategic Logic*

Created an intelligent agent capable of solving the classic logic game *Minesweeper* using Reinforcement Learning.

* **Approach:** Modeled the game board as a state-space environment where the agent learns optimal clicking strategies through reward-based feedback.
* **Technical Stack:** Implemented using Python (PyTorch) and Deep Q-Learning (DQN) concepts to achieve high win rates on varying difficulty levels.
